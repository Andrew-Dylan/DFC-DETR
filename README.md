# DFC-DETR
Dynamic Sparse Attention and Frequency-Spatial Complementary Network for SAR Ship Detection

# Experiments Environment
* **Operating System:** Ubuntu 22.04
* **GPU:** NVIDIA GeForce RTX 4090 24GB
* **CPU:** Intel(R) Xeon(R) Gold 6530
* **Python:** 3.11.8
* **PyTorch:** 2.2.2
* **torchvision:** 0.17.2+cu121
* **mmcv:** >=2.0.0
* **timm:** 1.0.7

# Environment Configuration
To set up the environment, please run the following command:
* `python setup.py develop`

# Dataset Downloads:
The datasets used for training and validation are available at the following links:
* **SSDD Dataset(8:2):** https://github.com/TianwenZhang0825/Official-SSDD
* **HRSID Dataset(6.5:3.5):** https://github.com/chaozhong2010/HRSID
